This is an AI chatbot designed with the aim of adding interactivity to the ai responses, users can collapse excessive content, so they easily keep track of points, buttons and links, so users can refer to resources, or use part of the response as a new prompt.

To run this project on local machine follow the steps;

1. run "git clone https://github.com/chukwuka4u/enama.git".
2. run "npm install".
3. visit https://www.openrouter.ai.
4. signup or login.
5. create and copy your API key, choose your prefered AI model string.
6. paste in .env file as, OPENROUTER_API_KEY and OPENROUTER_MODEL variables.

If you want to contribute to the project;

1. fork the project.
2. run "git clone #url" url is your forked repo url.
3. follow step 3 - 6 above.

Check the "Issues" tab for ways you can contribute (:
